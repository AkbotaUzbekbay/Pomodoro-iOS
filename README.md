# Pomodoro iOS

Simple Pomodoro timer app built with SwiftUI.

## Features
- Start / pause timer
- Reset timer
- Custom focus and break time
- Category selection
- Session history

## Screens
- Main screen with timer
- Settings screen to adjust time
- History screen with completed sessions

## Tech
- SwiftUI
- Timer
- State management (ObservableObject)

## How it works
- User sets focus and break time in Settings
- Starts timer on main screen
- After session completes, it is saved to history
